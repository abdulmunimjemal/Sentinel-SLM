# Agent Instructions

## Python Environment
ALWAYS use the virtual environment located at `.venv`.
- Python executable: `.venv/bin/python`
- Pip executable: `.venv/bin/pip`

When running python scripts, use:
`.venv/bin/python scripts/your_script.py`

When installing packages, use:
`.venv/bin/pip install package_name`

## Git
This is a git repository. Ensure `.gitignore` is properly configured to exclude data and venv files.
